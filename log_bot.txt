2023-01-29 21:07:43.609869 Answer:

Для отправки сообщения на электронную почту в пайтон вам понадобится модуль smtplib. Этот модуль позволяет программистам отправлять электронные письма при помощи протокола SMTP.

Ниже приведен пример кода для отправки сообщения на электронную почту с помощью модуля smtplib:

import smtplib

# Укажите адрес и порт вашего SMTP-сервера
smtp_server = "smtp.example.com"
smtp_port = 587

# Укажите адрес электронной почты и пароль от него
sender_email = "name@example.com"
password = "password"

# Укажите адрес электронной почты получателя
receiver_email = "name@example.com"

# Создайте сообщение
message = """\
Subject: Тестовое письмо

Это тестовое письмо из Python."""

# Отправьте сообщение
with smtplib.SMTP(smtp_server, smtp_port) as server:
    server.ehlo()
    server.starttls()
    server.ehlo()
    server.login(sender_email, password)
    server.sendmail(sender_email, receiver_email, message)

print('Сообщение отправлено')
